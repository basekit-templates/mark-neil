Mark Neil Portfolio
===================

BaseKit's Mark Neil Portfolio Template

RECOMMENDED IMAGE SIZE FOR HOMEPAGE SLIDESHOW: 

The image size that works best on the slideshow gallery is 960x600px. 

If you use a bigger image it can be cropped to 960x600px by selecting the option 'crop' on the editor.

Images smaller than 960x600px won't fill the width or height of the slideshow.
